Project Submission Proposal

Title:

HiSense: AI-powered real time object detection assistant for the Visually Impaired Persons (VIPs) - (Prototype)

Problem Statements:

A lot of people are visually impaired, and they face challenges every day to detect close objects or walk around safely and independently. The traditional aids canes or someone holding them along to help the VIPs but might not necessary interpret surrounding to them in real time. And some without any help at all.

Proposed Solution:

HiSense is a web-based AI assistant that uses real-time object detection with voice command to detect and announce the close objects to the user. The App also listens for voice commands like “Start”, “Stop”, and “Thank You” for now which we intend to add more voice command to better assist the VIPs. The App open webcam of computer or tablet or phone camera to feed the AI model and responds with audio feedback to guide the user safely. Currently the app can recognize 20 Images like: Person, Bicycle, Car, Apple, Motorcycle,

Impacts:

  • Promotes mobility and independence for visually impaired persons
  • Designed to work with basic internet access via a web browser
  • Deployable on mobile devices in the future

Technical Overview:

  • Model: YOLOv8 for real-time object detection and Custom Trained Model
  • Backend: Flask (Python)
  • Frontend: HTML, JavaScript, and Bootstrap
  • Speech-to-Text: (Web Speech API) for voice commands
  • Text-to Speech: (Brower API) for responses
  • Deployment: Hosted on render.com
